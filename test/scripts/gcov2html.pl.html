NAME

gcov2html.pl - Convert the result of the GNU coverage testing tool to HTML format

SYNOPSIS

gcov2html.pl [Option]... input-file

Options:

[-v] [-version] [-h] [--help]

DESCRIPTION

gcov2html.pl converts the result of the GNU coverage testing tool (gcov) to HTML format. The result is written to STDOUT.

Arguments

input-file

Name of input file.

Options

-v
-version

Print version and exit..

-h
--help

Print usage information and terminate program.

DIAGNOSTICS

Warnings and error messages are directed to STDERR. There may be error messages from the Perl I/O system.

RETURN VALUE

Returns zero in case of normal successful completion. In case of an error returns a value different from zero, normally the value of errno (see PERL manual pages on "die").

BUGS

Not yet known. If you encounter any error please report it to the author with a brief description of the circumstances.

SEE ALSO

The documentation of the GNU coverage testing tool ("man 1 gcov" and "man 1 gcc").
